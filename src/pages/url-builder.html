The URL builder

The URL builder offers a fluent and comprehensive api to build google SERP URLs. 

$googleUrl = new GoogleUrl();
$googleUrl->setSearchTerm("Simpsons");

echo $googleUrl->getUrl();
// you can also write echo $googleUrl;

// https://www.google.com/search?lr=lang_en&hl=en&q=Simpsons&complete=0&pws=0

Change the language

You can choose what change the query language. This will affect local/lang results 

// english results
$googleUrl->setLang("en");
echo $googleUrl->getUrl();
// https://www.google.com/search?lr=lang_en&hl=en&q=Simpsons&complete=0&pws=0

// german results
$googleUrl->setLang("de");
echo $googleUrl->getUrl();
// https://www.google.de/search?lr=lang_de&hl=de&q=Simpsons&complete=0&pws=0

Change the TLD

Setting the lang also affects the TLD, but you can manually set a TLD 

// english results on google.co.uk
$googleUrl->setLang("en");
$googleUrl->setTld("co.uk");
echo $googleUrl->getUrl();
// https://www.google.co.uk/search?lr=lang_en&hl=en&q=Simpsons&complete=0&pws=0

// german results on google.com
$googleUrl->setLang("de");
$googleUrl->setTld("com");
echo $googleUrl->getUrl();
// https://www.google.com/search?lr=lang_de&hl=de&q=Simpsons&complete=0&pws=0

//reset the TLD
$googleUrl->setTld(null);
echo $googleUrl->getUrl();
// https://www.google.de/search?lr=lang_de&hl=de&q=Simpsons&complete=0&pws=0

Change Number of results per page

By default google shows 10 results per page. You can change this limit from 1 to 100 

// 20 results per pages
$googleUrl->setNumberResults(20);
echo $googleUrl->getUrl();
// https://www.google.de/search?lr=lang_de&hl=de&q=Simpsons&num=20&complete=0&pws=0

Change the Page

You can change the page to query. If you dont specify it, you will ever get the first result page 

// get the results on the second page (0 indexed)
$googleUrl->setPage(1);
// and reset the number result to 10
$googleUrl->setNumberResults(10);
echo $googleUrl->getUrl();
// https://www.google.de/search?lr=lang_de&hl=de&q=Simpsons&start=10&complete=0&pws=0
